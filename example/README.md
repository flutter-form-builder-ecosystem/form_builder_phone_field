# Example Project for form_builder_phone_field

This project demonstrates how to use the `form_builder_phone_field` package in a Flutter application.

## Getting Started

To get started with this example project, follow these steps:

1. **Clone the repository**:
   
   ```bash
   git clone <repository-url>
   cd form_builder_phone_field/example
   ```

2. **Install dependencies**:
   
   ```bash
   flutter pub get
   ```

3. **Run the application**:
   
   ```bash
   flutter run
   ```

## Features

- Demonstrates how to implement phone number input fields using the `form_builder_phone_field` package.
- Supports validation and formatting of phone numbers.
